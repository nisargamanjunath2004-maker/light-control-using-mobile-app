# light-control-using-mobile-app

The IoT Light Control via Mobile App is a smart automation project used to control lights wirelessly using a smartphone. In this project, a ESP8266 NodeMCU is connected to an LED or light source and linked to a mobile application such as Blynk through Wi-Fi.
When the user presses the ON or OFF button in the mobile app, the command is sent through the internet/Wi-Fi to the ESP8266. The controller receives the signal and switches the light ON or OFF accordingly. This project demonstrates the concept of IoT, wireless communication, and smart home automation.
Components Used – Short Note
ESP8266 NodeMCU – Controls the system and provides Wi-Fi connectivity.
LED – Acts as the light source.
220-ohm resistor – Protects the LED from excess current.
Breadboard – Used for circuit connections.
Jumper wires – Connect different components.
Smartphone – Used to control the light remotely.
Blynk – Mobile app used for controlling the LED/light.
